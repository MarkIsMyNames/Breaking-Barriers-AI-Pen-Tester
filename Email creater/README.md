# Qwen3-8B GUI Interface

Simple web GUI for the Qwen3-8B AI model.

## Setup

Install dependencies:
```bash
pip install -r requirements.txt
```

## Run

```bash
python gui_app.py
```

Then open `http://localhost:7860` in your browser.

## Usage

1. Enter your prompt
2. Click "Generate Response"
3. Adjust settings in "Advanced Settings" if needed

That's it!
